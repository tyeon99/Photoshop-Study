◈사진합성 시  주의할 사항 

①소스끼리 빛과 그림자의 방향 일치!!

   ▶빛의 방향이 다르면 : Transform (Filip,Skew,Rotat ..)시켜 통일

   ▶외부의 그림자가 없을 경우 : Brush, Gradient, feather ..등으로 생성

   ▶내부의 그림자가 없을 경우 : Brush ,LayerStyle..등으로 생성

   ★브러시는 Opacity(힘)을 10%내외로 약하게 사용하여 덧칠하여 사용한다.



② 톤의 일치!!

  ▶Image>Adjustment>Match Color(닮고싶은 파일 또는 레이어 선택)

  ▶색(Ctrl U/Ctrl B), 채도(Ctrl U/ Vibrance), 명도(Ctrl U) , 대비(Ctrl L/Ctrl M)



③ 소스 추출!!(일본말로 누끼)

  ▶펜툴로 깨끗하게

  ▶W:퀵셀렉션으로 테두리 자연스럽게 추출하기



④ 최종이미지 진하고 선명하게 하기

a.전체 합친 레이어 복사본 만들기 :♣Ctrl Shift Alt E

b.기록을 남기고 싶으면 Smart object

c.Filter > Other > High pass(면=50%회색/가장자리=검정흰색으로 강조되는 효과)

d.Blend 블랜드(합성)하기: Overlay(대비합성)

   (Overlay원리:50%회색은 적용안되며, 보다 밝은덴 더 밝게↑ 어두운덴 더 어둡게↓ 합성됨)





▶구름 브러쉬 '찾기':cloud brush photoshop 검색후 다운받아 설치 후 사용

▶구름 브러쉬 '등록':

- select 메뉴 Color Range(색범위) 로 흰구름 선택 후  Ctrl J 로 추출

- Ctrl I 색 반전 : 흰색일수록 투명 검은색일수록 진하게 등록됨

- Edit > Define Brush : 등록하여 (브러시세트로 저장 가능)사용



♣Ctrl J : 제자리 복사

♣Ctrl Shift J: 제자리 분리

♣Ctrl E : 선택 레이어끼리 합치기

♣Ctrl Alt E: 선택한 레이어의 합친 복사본 생성

♣Ctrl Shift E: 눈뜬 레이어 전체 합치기

♣Ctrl Shift Alt E: 눈뜬 레이어 전체의 합친 복사본 생성
