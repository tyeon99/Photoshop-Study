<피부 보정>
1. open
2. 반창꼬(spot healing brush tool) 클릭
3. ctrl + j
4. 스마트 오브젝트로 변경
5. ctrl + j
6. 레이어이름 변경 = vivid-high
7. 아래 레이어 선택한 후 레이어이름 변경 = gaussian blur
8. 필터메뉴 - blur - gaussian blur 클릭 ( 수치는 잡티가 안보일정도 까지만 준다 )
9. 맨위에 레이어 선택 후 브랜드모드 - vivid light 클릭
10. 필터메뉴 - other - high pass 클릭 ( 수치는 안보일 정도로 준다 )
* 이미지가 흐릿하게 나와야 한다 *
11. 아래레이어까지 선택 후 그룹을 만든다
12. alt키를 누르고 마스크 누른다 ( 검정색 마스크로 된다 )
13. 변경색 배경색을 흰색 검정색으로 변경
14. 브러쉬 선택 한 후 동그라미 크기 조절
15. 얼굴 전체를 문지른다 ( 눈썹 입술 제외 )
* 브러쉬로 칠할 때 검정색이 나오면 수치를 다운시켜야한다 *
